# PharmaTrack — Attendance Monitoring System

A QR-based attendance tracking system for the University of San Agustin College of Pharmacy.

---

## What it does

PharmaTrack replaces manual attendance sheets with a fast, QR-based check-in flow. Faculty generate a session QR code at the start of class. Students scan it to mark their attendance. Records are stored instantly and available for reporting.

---

## Who uses it

| Role | What they can do |
|------|-----------------|
| **Student** | Scan QR to check in, view personal attendance records |
| **Faculty** | Generate QR sessions, monitor class attendance |
| **Admin** | Manage users, view system-wide attendance, export reports |

---

## How it works

1. Faculty opens a session for their class
2. A QR code is displayed — valid for that session only
3. Students scan the code to check in
4. Attendance is recorded automatically with a timestamp
5. Late arrivals are flagged accordingly
6. Faculty and Admin can view and export reports
