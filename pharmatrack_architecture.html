01 ГўвӮ¬вҖқ System-Level Architecture
Frontend & Framework
Next.js 15.5.19 App Router with server-side rendering and API routes. TypeScript end-to-end. Client components use React 18 with Suspense boundaries and skeleton loaders. Realtime updates via Supabase Postgres Change subscriptions.
Next.js 15.5React 18TypeScript 5App Router
Database & Auth
Supabase (PostgreSQL) with Row Level Security on all 7 tables. Supabase Auth handles JWT sessions via httpOnly cookie (pharmatrack_token). Auth emails (signup confirmation, password reset) route through MailerSend SMTP. Service-role client used only in API routes ГўвӮ¬вҖқ never exposed to the browser.
Supabase PostgreSQLRLSMailerSend SMTPJWT httpOnly
Rate Limiting
Next.js middleware intercepts four API path prefixes. Upstash Redis sliding window ГўвӮ¬вҖқ distributed across all Vercel function instances. Fail-open: if Redis is unreachable, falls back to in-memory Map so scan traffic is never blocked by a Redis outage.
Upstash RedisSliding WindowFail-open
Monitoring & Performance
Sentry integrated (client + server + edge configs). Production-only, sendDefaultPii: false, 10% transaction sampling. 7 PostgreSQL indexes applied on hot columns: attendance_records (student_id, event_id, session_id, event_id+status, created_at), events (date), users (account_type).
Sentry7 DB IndexesRealtime Subscriptions
Email Infrastructure
MailerSend ($7/mo Starter) handles all transactional email via nodemailer SMTP relay. Event broadcasts go out in batches of 100 (8 SMTP sends for 800 students). Fire-and-forget: API returns after DB insert, emails go out async. Auth emails (confirmation, password reset) route through Supabase custom SMTP via MailerSend ГўвӮ¬вҖқ removing the 3 email/hr free-tier cap. 5,000 emails/mo included; est. ~4,000/mo at 5 events. Supports 800-student bulk onboarding in one day.
MailerSend SMTPnodemailer poolFire-and-forget
QR Code
qrcode.react (SVG, Error Correction Level H) for display. html5-qrcode for camera scanning. QR IDs are random 8-char hex (QR-XXXXXXXX) ГўвӮ¬вҖқ not derived from user ID, not predictable. PNG export via canvas clone. DOM initialization uses double requestAnimationFrame to prevent race conditions on slow mobile devices.
